Regresión Lineal

Es un modelo matemático usado para aproximar la relación de dependencia entre una variable Y (label) y una variable X (feature).

  • Modelo. Define la relación entre features y label. Una vez que lo construyamos con datos historicos vamos a predecir el futuro y permitir que la maquina pueda tomar esas decisiones con lo ya aprendido.
  • Training. En este proceso le daremos un dataset al modelo para que aprenda con datos que tienen un label. En esta etapa, la maquina aprendera como se relacionan las variables.
  • Inferencia. En esta etapa nuestro modelo ya puede realizar predicciones.
